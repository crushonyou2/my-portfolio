# 🧑‍💻 조지관 포트폴리오

React + Tailwind CSS 기반으로 제작된 개인 포트폴리오 사이트입니다.  
Spring Boot 백엔드와 데이터 설계, 시스템 운영, 소프트웨어 품질 검증 경험을 한 페이지에서 확인할 수 있도록 구성했습니다.

👉 **포트폴리오 바로가기:**  
🔗 https://crushonyou2.github.io/my-portfolio/

---

## 🚀 포트폴리오 개요

이 사이트는 다음과 같은 목표로 제작되었습니다:

- 백엔드 기반 IT 시스템 역량과 QA/TestOps 관점을 함께 보여주기
- 주요 프로젝트의 역할·측정 결과·운영 경험 정리
- 한국어/영어(i18n) 다국어 지원
- 문의 기능 및 GitHub 연결 등 취업용 기본 정보 제공

---

## 🧩 주요 기능

- 👤 **About** — 자기소개 및 핵심 역량 정리  
- 📘 **Projects** — 주요 프로젝트 설명 및 GitHub 바로가기
- 📞 **Contact** — 이메일 및 GitHub 연결
- 🌐 **다국어 지원** — 한국어 / 영어 토글  
- 🖥️ **반응형 UI** 및 애니메이션 제공  

---

## 📂 디렉토리 구조

```plaintext

```
