# 🧑‍💻 조지관 포트폴리오

React + Tailwind CSS 기반으로 제작된 개인 포트폴리오 사이트입니다.  
Java/Spring 중심의 백엔드 개발 경험과 프로젝트 기록을 하나의 페이지에서 확인할 수 있도록 구성했습니다.

👉 **포트폴리오 바로가기:**  
🔗 https://crushonyou2.github.io/my-portfolio/

---

## 🚀 포트폴리오 개요

이 사이트는 다음과 같은 목표로 제작되었습니다:

- **백엔드 중심 개발 역량**을 직관적으로 보여주기
- 주요 프로젝트(GildongE, Movie Diary 등)의 역할 및 기술 스택 정리
- 한국어/영어(i18n) 다국어 지원
- 문의 기능 및 GitHub 연결 등 취업용 기본 정보 제공

---

## 🛠️ 사용 기술

### **Frontend**
- React  
- Tailwind CSS  
- Framer Motion  
- i18next (다국어 지원)  

### **Backend (역량 기반 기술)**
- Java / Spring Boot  
- Spring Data JPA  
- MySQL / MongoDB  
- OAuth2 / JWT  

### **Infra / DevOps**
- Git & GitHub  
- AWS (EC2, S3 등 실 프로젝트 경험)  
- Docker  

### **Etc**
- EmailJS (문의 폼 처리)

---

## 🧩 주요 기능

- 👤 **About** — 자기소개 및 핵심 역량 정리  
- 📘 **Projects** — 주요 프로젝트 설명 및 GitHub 바로가기
- 📞 **Contact** — EmailJS 기반 문의 기능  
- 🌐 **다국어 지원** — 한국어 / 영어 토글  
- 🖥️ **반응형 UI** 및 애니메이션 제공
