# 🧑‍💻 조지관 포트폴리오

React + Tailwind CSS 기반으로 제작된 개인 포트폴리오 사이트입니다.  
여러 프로젝트에서 경험한 서버 운영, 데이터베이스 관리, AWS 기반 배포, 자동화 환경 구축 등을 한 페이지에서 확인할 수 있도록 구성했습니다.

👉 **포트폴리오 바로가기:**  
🔗 https://crushonyou2.github.io/my-portfolio/

---

## 🚀 포트폴리오 개요

이 사이트는 다음과 같은 목표로 제작되었습니다:

- 시스템 운영·서버·DB 중심의 기술 역량을 명확하게 보여주기
- 주요 프로젝트의 역할 및 및 운영 경험 정리
- 한국어/영어(i18n) 다국어 지원
- 문의 기능 및 GitHub 연결 등 취업용 기본 정보 제공

---

## 🧩 주요 기능

- 👤 **About** — 자기소개 및 핵심 역량 정리  
- 📘 **Projects** — 주요 프로젝트 설명 및 GitHub 바로가기
- 📞 **Contact** — EmailJS 기반 문의 기능  
- 🌐 **다국어 지원** — 한국어 / 영어 토글  
- 🖥️ **반응형 UI** 및 애니메이션 제공  

---

## 📂 디렉토리 구조

```plaintext
src/
├── components/          # About, Info, Contact, Project 컴포넌트
├── assets/              # 이미지, 포스터 파일

├── App.js               # 메인 페이지 구성
└── index.js             # 엔트리 포인트
```
